## 📖 Game Description

기차 안에서 카페 메뉴를 제작하면서 스토리를 진행하고,  
NPC와 상호작용 및 아이템 수집을 통해  
퀘스트를 해결하는 **2D 비주얼노벨 시뮬레이션 게임**입니다.

<br><br><br>

## ⚙️ Rule Setting

### Naming Rule
- Function, Class : PascalCase  
- 변수명 : camelCase  
- 파일 이름 : UpperCamelCase  

### Git & Branch
- 기능 단위 branch 생성  
- branch : `feature/작업자/기능`  

### Commit Convention
- feat : 새로운 기능 추가  
- fix : 버그 수정  
- refactor : 코드 리팩토링  
- chore : 동작에 영향을 주지 않는 사항 수정
- body : 구현한 기능 설명

### Communication
- Discord  
- Notion : [Travel Around The World 노션 페이지 바로가기](https://petite-freesia-1f0.notion.site/Travel-Around-The-World-4bca6f307b844572885ad153e1cc7a7b)
